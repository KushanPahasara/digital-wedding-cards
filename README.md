# Wedding Registry & Planner App

A modern, full-stack wedding planning application built with React 19, Tailwind CSS, and Motion.

## Features
- RSVP Management
- Guest List Tracking
- Wedding Registry Gallery
- Planner Dashboard
- Countdown Timer
- Memory Photo Gallery

## Prerequisites
- [Node.js](https://nodejs.org/) (Version 18 or higher recommended)
- npm (comes with Node.js)

## Getting Started

### 1. Download the Project
- In Google AI Studio, go to the **Settings** menu and select **Export to ZIP**.
- Alternatively, you can export directly to a **GitHub** repository.

### 2. Local Setup
Extract the ZIP file and navigate to the project directory in your terminal:

```bash
cd your-project-folder
```

### 3. Install Dependencies
Run the following command to install the required packages:

```bash
npm install
```

### 4. Run the Development Server
Start the application locally:

```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

### 5. Build for Production
To create a production-ready build:

```bash
npm run build
```
The output will be in the `dist` folder.

## Tech Stack
- **Framework:** React 19
- **Styling:** Tailwind CSS 4
- **Animation:** Motion (formerly Framer Motion)
- **Icons:** Lucide React
- **Routing:** React Router 7
- **Tooling:** Vite
